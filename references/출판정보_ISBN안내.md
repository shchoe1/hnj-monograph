# 출판 정보 · ISBN 확보 안내 (판권면 채우기)

작성: 2026-07 · 대상: 최성화 교수(발행 준비)
관련 파일: `latex/himnoejang-book.tex`(판권면/간기), `latex/himnoejang-cover-preface.tex`(판권 verso)

> **중요.** ISBN(및 전자책 eISBN)은 **임의로 만들 수 없는 공식 식별번호**입니다. 국립중앙도서관 한국서지표준센터가 **등록된 발행자(출판사)** 에게 발급하며, 지어낸 번호는 무효이거나 실제 다른 도서와 충돌합니다. 그래서 LaTeX 판권면에는 실제 번호 대신 **자리표시자 〔 〕** 를 넣었고, 아래 절차로 실번호를 받아 그 자리에 기재하시면 됩니다.

## 1 · 종이책 ISBN 받는 절차 (한국)

1. **출판사 신고** — 관할 시·군·구청에 출판사(발행자) 신고 → **신고확인증(출판등록번호)** 수령. (1인 출판도 가능)
2. **발행자번호 신청** — 서지정보유통지원시스템 **seoji.nl.go.kr** 가입 후 **발행자번호(출판사 고유번호)** 신청. (무료)
3. **도서별 ISBN 신청** — 발간 예정 도서마다 ISBN 신청 → **13자리 종이책 ISBN** 발급(무료). 형식 예: `978-89-발행자번호-서명번호-체크숫자` 또는 `979-11-…`.
4. **부가기호·바코드** — ISBN + 5자리 부가기호로 **바코드** 생성(정가 포함 여부 등). 뒤표지에 인쇄(인쇄소가 처리).
5. **(선택) CIP** — 출판시도서목록(CIP)도 같은 시스템에서 신청 가능.
6. **납본** — 발행 후 국립중앙도서관에 납본.

## 2 · 전자책(인터넷판) 번호

- **전자책 ISBN(eISBN)** — 종이책과 **별도의 13자리 ISBN**을 같은 시스템(seoji.nl.go.kr)에서 신청. 전자책은 형태(EPUB/PDF 등)에 따라 번호를 따로 받습니다. → 판권면의 「전자책 ISBN」 자리에 기재.
- **(선택) 웹/온라인 인용용 DOI** — 온라인판을 학술적으로 인용 가능하게 하려면 CrossRef 회원(기관/출판사)을 통해 DOI를 발급받을 수 있습니다(선택 사항, ISBN과 별개).
- 현재 온라인 내비게이터: <https://shchoe1.github.io/hnj-monograph/> (GitHub Pages) — 여기에 eISBN·저작권 표기를 추가하려면 말씀해 주세요.

## 3 · 판권면 자리표시자 → 채울 값 체크리스트

`latex/himnoejang-book.tex` 하단 판권면(간기)의 〔 〕 를 아래 값으로 교체하세요.

- [ ] **1판 1쇄 발행일** — 〔20__년 __월 __일〕
- [ ] **펴낸이(발행인)** — 〔이름〕
- [ ] **펴낸곳(출판사명)** — 〔상호〕
- [ ] **출판등록** — 〔제 ____ 호 · 등록일 ____.__.__〕
- [ ] **주소 / 전화 / 전자우편**
- [ ] **편집·디자인 / 인쇄·제본** 담당·업체
- [ ] **정가(값)** — 〔__,___원〕
- [ ] **종이책 ISBN** — 〔978-89-… 또는 979-11-… 13자리〕
- [ ] **전자책 ISBN(eISBN)** — 〔13자리〕
- [ ] (확인) 저작권 문구·일반식품 고지·AI 작성/저자 감수 문구 — 이미 기재됨

> 값을 정하시면, 알려 주세요 — LaTeX 판권면에 바로 반영해 드립니다. (지금은 임의 생성 금지 원칙에 따라 자리표시자로 두었습니다.)
